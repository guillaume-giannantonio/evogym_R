<h1 id="introduction">🌟 Introduction</h1>
<p>EvoGym is an imaginary gym website created using TypeScript, React and TailwindCSS! This project serves as a showcase for a fictional fitness center, demonstrating how modern web technologies can be utilized to create an engaging and interactive online presence.</p>

<h1 id="considerations">⚠️ Considerations</h1>
<p>EvoGym has been meticulously crafted to showcase the capabilities of TypeScript and React for gym and fitness websites. It stands as a creative playground for exploring web development in the context of fitness centers. While it offers a fully functional and visually appealing template, it's crucial to recognize its purpose as a foundation for your gym website projects.
<h3>A Creative Canvas</h3>
EvoGym is not intended to be a production-ready gym website for real-world use. Instead, it is designed to inspire and empower you to create your own unique gym website tailored precisely to your requirements and vision.
</p>

<h1 id="installation">⚙️ Installation</h1>
<p>Simply clone this project using the git clone commande :

```bash
git clone git@github.com:guillaume-giannantonio/evogym_TR.git
```

And install the dependencies using npm :
```bash
npm install
```
